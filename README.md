# Uncertainty Quantification for LLM-based Code Generation

## Getting Started
1. Get prompts and sample responses (infer.sh, inference.py)
2. Parse programs into ASTs (parse_results.py, ast_helper.py)
3. Verify correctness of responses (verification.py)
4. Conduct RisCoSet method (method.py, optimize.py)
5. Evaluate results (evaluate_result.py)
